자바스크립트 변수

변수명 작성 방법

1. 대문자와 소문자를 구분한다.
2. 문자로 시작해야 한다. 특수문자는 '_'만 가능하다.
3. 자바스크립트와 약속된 예약어(명령어)는 변수명으로 사용할 수 없다.

범수의 범위

1. 전역 변수는 자바 스크립트의 최상위 객체인 window 객체에 포함되는 변수로
   모든 함수에서 공통적으로 사용할 수 있다.
   변수에 저장된 값이 계속 유지된다.
   
2. 지역변수는 함수나 객체 내부에서 선언하고 함수나 객체가 종료되는 시점에 메모리에서
   소멸된다.
   

자바스크립트 데이터 타입

1. 문자(string)
2. 숫자(number)
3. 논리(boolean)
4. null
5. 객체(object)
6. 함수(function)

변수의 선언과 저장

1. var, let, const 예약어로 변수를 선언한다. 아무것도 안붙여도 변수는 선언된다.
2. let, const는 ES6부터 추가되었다.
3. 변수의 데이터 타입은 변수에 저장되는 데이터의 타입에 따라 자동으로 결정된다.
